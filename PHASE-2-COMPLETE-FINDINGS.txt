================================================================================
                          CRITICAL FINDINGS (12)
================================================================================

SECURITY VULNERABILITIES (3):
────────────────────────────────
1. CLI-CRIT-2: HTTP Header Injection (CRLF)
   - Token unsanitized in Authorization header
   - Attack: GITHUB_TOKEN="valid\r\nX-Injected: value"
   - Impact: Auth bypass, response smuggling, cache poisoning
   - Fix: 1 hour

2. CLI-CRIT-3: SSRF Vulnerability (Server-Side Request Forgery)
   - No URL protocol or hostname validation
   - Attack: file:///etc/passwd, https://localhost:8080, https://192.168.1.1
   - Impact: Read arbitrary files, access internal services, RCE
   - Fix: 2 hours

3. CLI-HIGH-8: No Checksum Verification
   - Downloaded artifacts not validated
   - Impact: Accept poisoned/corrupted files from MITM attack
   - Fix: 2 hours

DATA LOSS RISKS (5):
────────────────────────────────
4. M-CRIT-1: Race Conditions in Cache
   - Concurrent Maven builds corrupt cache
   - No file locking mechanism
   - Impact: Data corruption, inconsistent test ordering
   - Fix: 4-6 hours

5. INT-CRIT-2: Disk Full Silent Corruption
   - Cache file left in corrupted state when disk fills
   - No atomic writes or rollback
   - Impact: Silent data loss, impossible recovery
   - Fix: 3-4 hours

6. G-CRIT-5: Multi-Project State Corruption
   - Subprojects share state file instead of isolation
   - Projects overwrite each other's data
   - Impact: Data corruption in monorepos
   - Fix: 3 hours

7. INT-CRIT-1: No Cache Coordination
   - Maven and Gradle both write to .test-order/
   - No coordination during parallel builds
   - Impact: Cache inconsistency, race conditions
   - Fix: 4 hours

8. INT-CRIT-3: No Version Protocol
   - No cache format version marker
   - Undefined behavior on upgrades/downgrades
   - Impact: Silent data corruption, migration impossible
   - Fix: 2-3 hours

FEATURE FAILURES (4):
────────────────────────────────
9. G-CRIT-4: Gradle Core Features Broken
   - Learn mode doesn't generate dependency index
   - Order mode doesn't reorder tests
   - State file not created
   - Impact: PRIMARY PLUGIN FEATURE NON-FUNCTIONAL
   - Fix: 4-8 hours

10. G-CRIT-1: Java 26 Class File Version
    - Plugin compiled with bytecode v70 (Java 26)
    - Gradle 8.14 only supports up to v65 (Java 21)
    - Impact: Plugin completely unusable on Java 26+
    - Fix: 1 hour

11. M-CRIT-1: Maven Silently Ignores Changed Files
    - Non-existent files in 'changed' parameter silently ignored
    - Invalid configs not detected at startup
    - Impact: Silent failures, CI pipeline inconsistency
    - Fix: 2 hours

12. M-HIGH-9: Plugin Not Found in Reactor Root
    - mvn test-order:show-order fails in reactor root
    - Error: "No plugin found for prefix 'test-order'"
    - Impact: Cannot run from multi-module project root
    - Fix: 1 hour
